# 📚 Projet Lycée : Aventure Textuelle (C)

### Description
Projet réalisé dans le cadre de ma formation en **Terminale STI2D**. L'objectif était de créer un "livre dont vous êtes le héros" entièrement en **Langage C**. 

### Défis techniques
* Gestion des structures de données pour les choix du joueur.
* Utilisation de boucles et de conditions complexes pour la narration.
* Interface console interactive.
