# 🌐 Projet Nexus

### Description
Le projet Nexus est une réalisation axée sur un nouveau type de réseau social basé sur l'intelligence artificielle. Ce projet m'a permis d'approfondir mes connaissances en JavaScript, HTML et CSS.

### Points clés
- **Objectif :** Créer une plateforme de communication
- **Technologies :** Python, API, HTML, CSS, JavaScript, etc.
- **Défi technique :** La gestion des données en temps réel
